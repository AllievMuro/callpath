The Problem

Decoding Base transactions is painful by default

  • Raw debug_traceTransaction output is hex soup — unreadable without an ABI.
  • ABI hunting across Basescan, IPFS, and GitHub takes hours per contract.
  • Unverified contracts return nothing — or crash your decoder entirely.
  • ERC-4337 UserOp bundles collapse multiple ops into one opaque blob.
  • Running an archival node + decoder stack costs thousands per month.

Capabilities

Everything decoded, nothing assumed

A single API surface covering verified contracts, unknown bytecode, account abstraction, and real-time streaming.

🔍

Full Call Tree Decoding

POST a tx hash, receive a structured JSON call tree with every internal call, depth, gas, and return value decoded from Basescan-verified ABIs.

🧬

Bytecode Fingerprinting

Unverified contracts get partial decoding via 4-byte selector matching. Every field carries a confidence score so you know exactly what to trust.

📦

ERC-4337 UserOp Unpacking

Bundled UserOps are exploded into individual decoded operations — each with its own gas accounting, decoded calldata, and revert attribution.

⚡

WebSocket Streaming

Subscribe to any contract address and receive decoded call events in real time via WebSocket — no polling, no archival node required.

📚

Live ABI Registry

ABIs are auto-indexed from Basescan every block. Community submissions via ERC-7730-compatible JSON pinned to IPFS are indexed within one block confirmation.

📊

Block-Range Batch Queries

Query up to 1000 blocks in a single request for historical analytics. Ideal for building dashboards, audit trails, and protocol-level reporting.

How It Works

From hash to human-readable in milliseconds

Callpath runs a live decode pipeline on Base, so you never touch an RPC node or manage ABIs yourself.

1

Submit a tx hash, block range, or contract address

REST or WebSocket. Any Base transaction, including contracts that aren't verified on Basescan.

2

ABI registry lookup + bytecode fallback

Verified ABI is fetched instantly. For unknown contracts, the 4-byte selector DB provides partial decoding with confidence scores.

3

ERC-4337 unpacking and event correlation

UserOp bundles are decomposed. Emitted events are matched to function calls. Revert reasons are decoded to strings.

4

Receive structured JSON — ready to render

No post-processing needed. Use it directly in your portfolio tracker, AI agent, or wallet UX layer.
